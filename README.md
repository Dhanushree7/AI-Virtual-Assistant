# AI Virtual Assistant

An advanced AI-powered Virtual Assistant built using:

- **Java (Spring Boot)** for the main backend
- **Python (FastAPI)** for the AI service
- **Groq API** for fast LLM responses
- **MySQL** for structured data storage
- **Future scope:** RAG, memory, voice, automation, and multi-agent features
